[React Transition Group](https://reactcommunity.org/react-transition-group/)

```jsx
npm install react-transition-group --save
```

좀더 깔끔하고 부드러운 CSS 애니메이션을 구사할때 좋은 라이브러리 이다.
그러나 지금 이 상황에서는 styled-components를 사용할때 과연 적용이 가능한지는 추후에 알아봐야 할 것 같다.
CSSTransition에 구성된 라이브러리 이다.

```jsx
import Transition from 'react-transition-group/Transition';
```

```jsx
<div
  style={{
    backgroundColor: 'red',
    width: 100,
    height: 100,
    margin: 'auto',
  }}></div>
```

이러 했던 코드를 아래 있는 코드로 바꿔 트렌지션을 설정 할 수 있다.

```jsx
<Transition in={this.state.showBlock} timeout={1000} mountOnEnter unmountOnExit>
  {(state) => (
    <div
      style={{
        backgroundColor: 'red',
        width: 100,
        height: 100,
        margin: 'auto',
        transition: 'opacity 1s ease-out',
        opacity: state === 'exiting' ? 0 : 1,
      }}
    />
  )}
</Transition>
```

in 이란, 어떻게 이벤트 헨들러를 선택하면 되며, timeout은 dealy시간이다.
mountOnEnter 은 자리공간을 없애기 위함이며, 뜻대로 마우스가 입력되면 실행되도록 하는 함수다.
